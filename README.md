# نظام تعويض القياسات الناقصة

نظام مستقل بالكامل (صفحة واحدة تعمل بدون إنترنت كـ PWA) لحساب **القياسات الناقصة**
تلقائياً من **القياسات المتوفرة** عبر معادلات قابلة للتعديل — مع إظهار **أي قياس تم تعديله**
و**ما هي المعادلة المستخدمة**.

## الفكرة

في شاشة القياسات، يُدخل المستخدم القياسات المتوفرة فقط ويترك الناقص فارغاً، ثم يضغط
**«استكمال الناقص»**. يقوم النظام بـ:

1. قراءة القياسات المتوفرة.
2. حساب كل قياس ناقص من معادلته الاحتياطية المبنية على القياسات الأخرى.
3. حلّ **التسلسل**: إذا اعتمد قياس ناقص على قياس ناقص آخر، يُحلّ بالترتيب الصحيح تلقائياً.
4. تمييز كل قياس محسوب بلون مختلف، وكتابة المعادلة أسفله.
5. عرض **تقرير** يوضّح لكل قياس: اسمه + المعادلة (الرمزية والرقمية) + القيمة الناتجة.

## المميزات

- **منتجات متعددة**: «الثوب السعودي» مُعرّف مسبقاً، مع إمكانية إضافة أي منتج آخر.
- **معادلات قابلة للتحرير بالكامل** من شاشة «القياسات والمعادلات».
- **محرّك تقييم آمن** للمعادلات (بدون `eval`) يدعم `+ - * / ( )` والأرقام العشرية ومفاتيح القياسات.
- **تنبيه واضح** عند تعذّر حساب قياس (نقص القياسات المطلوبة أو غياب المعادلة).
- **حفظ محلي** + تصدير/استيراد JSON.
- يعمل **بدون إنترنت** (Service Worker) وقابل للتثبيت على الجوال.

## المعادلات الافتراضية للثوب السعودي

> هذه قيم/نسب مبدئية قابلة للتعديل حسب نظام كل خياط من شاشة «القياسات والمعادلات».

| القياس | المفتاح | المعادلة الاحتياطية |
|--------|---------|----------------------|
| الطول | `length` | (أساسي) |
| الكتف | `shoulder` | `chest/3` |
| الصدر | `chest` | `waist + 6` |
| الوسط | `waist` | `chest - 6` |
| الكعب (الأسفل) | `hem` | `chest + 8` |
| طول الكم (الردن) | `sleeve` | `length - 18` |
| الزند | `bicep` | `chest/4` |
| الأسوارة | `cuff` | `neck + 1` |
| الرقبة | `neck` | `chest/6` |

## طريقة كتابة المعادلات

تُكتب المعادلة بمفاتيح القياسات الإنجليزية والعمليات الحسابية:

```
chest - 6
(chest / 2) + 5
length - 18
neck * 1.1
```

اترك المعادلة فارغة للقياسات الأساسية التي لا تُشتق من غيرها.

## التشغيل

افتح `index.html` في المتصفح مباشرة — لا يحتاج أي تثبيت أو خادم.

## البنية

- `index.html` — التطبيق كامل (واجهة + محرّك المعادلات).
- `manifest.json` / `sw.js` — دعم PWA والعمل دون اتصال.
